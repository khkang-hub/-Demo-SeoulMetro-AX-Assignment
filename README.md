# 서울교통공사 AX 홈페이지 리뉴얼 프로토타입

서울교통공사 공식 홈페이지를 시민 중심 관점으로 재구성한 과제 제출용 정적 웹 프로토타입입니다.

## 구성

- `index.html`: 메인 프로토타입
- `styles.css`: 반응형 UI 및 접근성 스타일
- `app.js`: 카드 검색, AI 도우미 시뮬레이션, 링크 렌더링
- `report.md`: 제출용 리포트 초안
- `.github/workflows/pages.yml`: GitHub Pages Actions 배포 예시
- `.nojekyll`: Jekyll 처리 비활성화

## 로컬 실행

`index.html` 파일을 브라우저로 열면 바로 실행됩니다.

## GitHub Pages 배포

1. GitHub 저장소 생성
2. 모든 파일을 저장소 루트에 업로드
3. `Settings > Pages`
4. Source를 `Deploy from a branch`로 선택
5. Branch `main`, Folder `/root` 선택 후 저장
6. 배포 완료 후 생성된 URL 제출

## 주의

이 프로토타입은 과제 제출용이며 실제 서울교통공사 공식 서비스가 아닙니다.
